Региональный аэропорт

Веб-приложение на Django для управления услугами регионального аэропорта. В системе есть роли администратора, сотрудника аэропорта и пассажира.

Функции:
- просмотр служб аэропорта и графика смен;
- регистрация пассажиров;
- оформление заявок на обслуживание;
- просмотр личных заявок пассажира;
- рабочий экран сотрудника со списком заявок на смену;
- административное управление сотрудниками, сменами и заявками.

Создание групп пользователей и тестовых данных:

```bash
python create_groups.py
python create_test_data.py
```
